# <p align="center">TaroParse taro版本富文本解析组件</p>
## <p align="center">支持Html及markdown转wxml可视化</p>
##### <p align="center"><code>版本号:1.0.0</code></p>
## 特色
## 特色
- 支持LaTex数学公式
- 支持yuml流程图
- 支持代码语法高亮、代码块行号显示
- 支持emoji表情😉
- 支持上标、下标、下划线、删除线、表格、视频、图片
- 支持typographer字符替换
- 支持多主题切换
- 支持Markdown TodoList
